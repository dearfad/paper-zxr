# MCF-7 / MDA-MB-231 3D类器官图像质控与预处理系统

## 📋 系统概述

本系统针对乳腺癌双细胞系（MCF-7: Luminal A型；MDA-MB-231: 三阴性）3D类器官培养项目，提供：

1. **人工初筛记录模块** - 标准化记录不合格图像及原因
2. **自动质控模块** - 基于拉普拉斯方差、亮度统计、划痕/气泡检测
3. **图像预处理模块** - 尺寸统一、CLAHE增强、去噪、对齐
4. **批量处理工作流** - 整合全流程，自动分类管理

## 🎯 质控标准（项目特定）

| 指标 | MCF-7标准 | MDA-MB-231标准 | 说明 |
|------|-----------|----------------|------|
| **清晰度** | 拉普拉斯方差 > 55 | 拉普拉斯方差 > 45 | MDA-MB-231边缘不规则，阈值放宽 |
| **亮度均值** | [50, 200] | [40, 220] | 避免过曝/欠曝 |
| **对比度** | 标准差 > 20 | 标准差 > 20 | 确保细胞边缘清晰 |
| **划痕** | 无严重亮线 | 无严重亮线 | 霍夫直线检测 |
| **气泡** | < 3个 | < 3个 | 圆形度>0.7的亮斑 |

## 🚀 快速开始

### 1. 环境准备

```bash
pip install opencv-python numpy pandas tqdm openpyxl
```

### 2. 批量处理（推荐）

```bash
# 处理全部图像
python run_qc_pipeline.py ./raw_images ./output

# 仅处理MCF-7细胞系
python run_qc_pipeline.py ./raw_images ./output --cell_line MCF7

# 仅处理MDA-MB-231细胞系（调整阈值适应其不规则形态）

```

### 3. 人工初筛记录

当人工检查图像发现问题时：

```bash
python image_qc_preprocessing.py --manual_reject \
    --image "./raw/MCF7_001_24h_F01.tif" \
    --cell_line "MCF7" \
    --sample_id "MCF7_001" \
    --timepoint "24h" \
    --view_id "F01" \
    --reason "划痕干扰" \
    --severity "high" \
    --operator "张实验员" \
    --notes "图像中央有水平亮线，疑似载玻片划痕"
```

## 🔧 核心功能详解

### 1. 自动质控算法

**拉普拉斯方差（清晰度）**

```python
from image_qc_preprocessing import AutoImageQC

qc = AutoImageQC()
variance = qc.calculate_laplacian_variance(image)
# variance > 50 为清晰
```

**亮度统计**

```python
stats = qc.calculate_brightness_stats(image)
# stats['mean'] 应在 [50, 200]
# stats['std'] 应 > 20
```

**划痕检测**

- 使用Canny边缘检测 + 霍夫直线检测
- 检测长度>50px的直线
- > 5条直线判定为"有划痕"

**气泡检测**

- 二值化阈值200检测亮斑
- 圆形度>0.7判定为气泡
- > 3个气泡标记为异常

### 2. 预处理流程

1. **中心裁剪** (0.8比例) - 聚焦培养核心区，排除边缘
2. **尺寸统一** (336×336px) - 适配CLIP/ViT模型输入
3. **CLAHE增强** - 自适应直方图均衡化，提升边缘对比度
4. **去噪** - 高斯模糊(σ=0.5) + 中值滤波(3×3)

### 3. 人工初筛记录

支持的不合格原因：

- 图像模糊
- 曝光过度/不足
- 划痕干扰
- 气泡干扰
- 杂质污染
- 视野偏移
- 细胞脱落
- 培养失败形态
- 其他

## ⚠️ 关键注意事项

### 细胞系特异性处理

**MCF-7特性**：

- 形成紧密球状结构，圆度高
- 边缘清晰，适合高清晰度阈值
- 直径范围：50-400μm

**MDA-MB-231特性**：

- 侵袭性生长，边缘不规则呈星状
- 可能有伪足突起，允许较低清晰度阈值
- 直径变异大：30-500μm
- 建议降低拉普拉斯阈值至45

### 命名规范检查

脚本会解析文件名格式：`{CellLine}_{SampleID}_{Timepoint}_{ViewID}.tif`

示例：

- ✅ `MCF7_001_24h_F01.tif`
- ✅ `MDA231_015_48h_F03.tif`
- ❌ `IMG_20240301_001.jpg` (无法解析)

## 🔬 与项目阶段衔接

### 第一阶段（基础筹备）

- 使用本系统建立图像质量标准
- 完成预实验图像的质控验证
- 确定两种细胞系的形态学金标准

### 第二阶段（数据基建）

- 批量处理时序图像（0h, 6h, 12h, 24h, 48h, 72h）
- 生成标准化数据集（训练/验证/测试集）
- 输出用于LLaVA-Med和传统ML模型的格式

### 第三阶段后（模型开发）

- 预处理后的图像直接输入模型训练
- 质控日志用于数据清洗报告
- 支持模型可解释性分析（对比原始与预处理图像）
